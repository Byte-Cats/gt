# gt
Go Terminal 
## Minimalist Terminal emulator written in Go

</div>


### Desired Feature Set

- Image Preview (PNG, JPEG, GIF, BMP, TIFF, WebP)
- Supports 256 colors
- Scrollable Text Buffer
- UTF-8 support


### Build from source
```bash
go mod init && go mod tidy && go build
```
